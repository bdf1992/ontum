# Sources — inspiration and direction, read-only

This directory holds external sources that are aligned with the work: things to
return to **when we hit walls or question our sanity**. Each entry records what
the source actually says, why it is aligned, and what direction it points —
separated, so the inspiration never gets confused with the claim.

Like `docs/phase-2/`, these are **read-only context**. The loop must not
import, integrate, or build from anything in here. A source justifies a
direction; it does not specify a build. If a build seems to need one of these,
that is a scope error and gets reported, not coded around.

- `compression-prediction-entropy.md` — prediction ≡ compression; the
  information-theoretic spine under the glyph-field direction.
- `rosetta-creole.md` — ontum is a living third language between code and
  models, not a finished translation key; what the language surface is *for*.
