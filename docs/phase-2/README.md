# Phase-2 material — read-only context

These documents describe the wider project the workflow engine exists to build:
the cellular language, the 27-cell sheaf, the Ontum unit system, and the
ontogrammatic typing discipline.

Per the Phase 1 briefing they are **read-only context** for the stage-0 work.
The loop skeleton must not import, integrate, refactor, or improve anything in
here. If a Phase 1 build seems to need them, that is a scope error and gets
reported, not coded around.

- `autojective-polysheaf.md` — Sites, Seams, the Atlas; the shaper before the shape.
- `cube-alphabet.md` — the cube/alphabet token-system mapping; the empty center.
- `ontogrammatic-systems.md` — the typing discipline for generated artifacts.
- `ontum-evolution.md` — the grip ledger: ontum/onta, keystone, cant, drift.

Note: the doctrine (§15–§16) references `docs/culture/the-idea.md` and
`docs/culture/the-ambient-loop.md`. Those files have not been provided yet —
absence noted, not invented.
