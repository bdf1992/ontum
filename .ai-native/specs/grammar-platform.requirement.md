# Requirement — the grammar platform adapts a target system

A target bounded environment is mapped onto the root behaviors (8 systems
× 8 pointers), and its topological placement (Internal/External ·
Ingress/Egress · Local/Global, measured against the Boundary) configures
the environment's setpoints.

## Acceptance condition

Given one real, non-ontum target system (the gallery is the first
candidate), the platform produces an accountable receipt that the target
was mapped onto the grammar and its setpoints were admitted (not code
constants).
