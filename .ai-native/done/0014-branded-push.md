Written before code, per §9.4. The watcher's first nomination, stamped
by bdo in conversation ("git push does"). When this line is met, stop.

> **Done when:** a session's commits leave the machine only through
> the branded push — the pen's `push` verb refuses the trunk, a dead
> branch (a merged PR from this head: a push can no longer strand by
> construction), and a red suite unless declared (`--red-ok`, with
> the declaration owed to the PR story at hand-off); force exists
> only as `--force-with-lease` on the session's own branch, plain
> `--force` never; raw `git push` is hook-denied with the message

> this increment itself reaches origin through the branded verb —
